Breadcrumbs

The breadcrumb classes use a structure of nav > ordered list > list item, the nav has an aria-label of 'breadcrumb', the ordered list has a class of 'breadcrumb' & the list item has a class of 'breadcrumb-item' with the class 'active' if it is the current/active page. The 'breadcrumb' class adds the padding & style for the overall box & the 'breadcrumb-item' adds the style for each breadcrumb.

Buttons

The class 'btn' adds the padding & style of the button, and the 'btn-x' adds the colour. The class 'btn-outline-x' will add the colour to the border/outline instead of the background.





Button Groups

You can use the class 'btn-group' to group buttons together. Add '-lg' or '-sm' onto the end of the group class to change the size. 'btn-group-vertical' will change the button layout to a vertical position. You can also nest 'btn-group' by using 'btn-toolbar' or just by placing a 'btn-group' within another to add a dropdown menu.

Carousel

The carousel class will easily allow you to create an image carousel complete with animations just using classes. All components will require to be nested within the 'Carousel' class to function properly. Add the class 'slide' for a smooth animation/transition.

Adding a caption on the image

Beneath the img tag add a div with the class 'carousel-caption', anything within that div will appear on the image instead of beneath.

The caption will appear beneath the slide if the class 'carousel-caption' isn't used.

To add a fade transition to slides, add the class 'carousel-fade' to the main div with the class 'carousel'.

Previous Next

Each list item will require its 'data-target' to be pointed to the id of the Carousel, and the ordered list tag will require the class 'carousel-indicators'.

Collapse

There are two different types of collapses: collapses & accordion, use the class 'collapse' on the collapsible elements for the collapse option, and place each accordion section within a single div. Each accordion section should have two child elements, one being the clickable heading and the second being the collapsible content.

Links target the collsapsed element using href

For both links and buttons you want to set the data-toggle attribute to "collsapse", set the aria-controls to the id of the collapsed element and set aria-expanded to "false".


Set the data-toggle to 'collapse' & the href to the id of collapsed content.
Add the id to the collapsed content, then add the class 'collapse'. The collapsed content and the clickable heading must be siblings within a div.
To have an item open by default add the class 'show' to the content and set aria-expand to 'true' for the clickable heading.

With column widths, you can control how big elements are based off of the screen size, the options are: col-x (extra small), col-sm-x, col-md-x, col-lg-x & col-xl-x

col-x: Screens smaller than 576px
col-sm-x: Screens larger than 576px
col-md-x: Screens larger than 768px
col-lg-x: Screens larger than 992px
col-xl-x: Screens larger than 1200px
The width of the parent element or screen is split up into 12 segments, you assign 'x' to the number of segments you want a element to take up. For example, if you wanted a navigation bar with four equal tabs on every screen above 576px, you would assign each tab the class 'col-sm-3'

If the number elements you want in a container don't equally add up to 12 (e.g 5 elements), then you can equally split the elements across all screen sizes using the class 'col', if you want the layout or number of columns per row to change based on screen size you can use the class 'col-y', 'y' being the screen size such as sm, md, etc.

Jumbotron

The jumbotron uses utility classes for typography and spacing to space content out within the larger container, add the class 'jumbotron-fluid' to fill up the whole width of the parent element.

jumbotron


jumbotron-fluid

Adding the 'jumbotron-fluid' class will make the jumbotron take up the full width of the parent element regardless of screen size. It will also remove some of the padding within the jumbotron. You will also want to nest the content in the jumbotron with a class of 'container-fluid' to make sure the content takes up the width of the jumbotron.

Modal

Modals will have two components, the trigger button and the modal div, they aren't required to be nested within the same div but instead use the data-target/href and id linking method. The modal will need to have a class of 'modal', then that will have a child div with the class 'modal-dialog' to center the modal, within that div there will be the child div with the class 'div-content'. This div contains the header, main body & footer for the modal, they have the classes 'modal-header', 'modal-body' & 'modal-footer' respectively.

'modal-body'

Center the Modal
Add the class 'modal-dialog-centered' to the div with the class 'modal-dialog' to center the modal vertically along the screen.


Scollable Modal Body
Add the class 'modal-dialog-scrollable' to the 'modal-dialog' div to make the body of the modal scrollable. If the content isn't long enough to overflow past the container then the scrollbar won't appear.
